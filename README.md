# discrod-vote-bot
work in progress  

Discordで簡易的なアンケートをとれるBOTを作成中  
# 使い方  
チャットルームにて「/vote」のコマンドの後に半角スペースを空けたのち、  
-質問内容  
-選択肢  
を半角スペースを空け送信します。  
例  
![サンプル](https://user-images.githubusercontent.com/42764831/55631152-561db380-57f2-11e9-931a-8ad7b23811ca.PNG)  
選択肢は現在a~eまでの5個まで増やすことができます。  
# 現状の問題点・改善点  
重複投票を防ぐ
